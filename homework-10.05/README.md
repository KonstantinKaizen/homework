

### Задание 1

Что такое балансировка нагрузки и зачем она нужна?

```
Балансировка нагрузки это процесс распределения нагрузки между серверами, для максимизации скорости и максимальной утилизации их ресурсов.
```

### Задание 2

Чем отличаются алгоритмы балансировки Round Robin и Weighted Round Robin? В каких случаях каждый из них лучше применять?

```
Round Robin - запросы распределяются по серверам последовательно, алгоритм применяется, когда все сервера одинаковой мощности и запросы имеют одинаковую сложность. 
Weighted Round Robin - тот же round robin, но мы можем указать "вес" для каждого сервера, так сервера которые будут иметь больший вес будут обрабатывать больше запросов.
```
